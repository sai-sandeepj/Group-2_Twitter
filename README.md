### Steps to run the application
#### Frontend
```sh
$ cd frontend
$ npm install
$ npm start
```
#### Backend
```sh
$ cd backend
$ npm install
$ node index.js
```
#### Kafka
```sh
$ cd kafka-backend 
$ npm install
$ node server.js
```
